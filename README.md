## 📁 Project Spotlights

### 1. AI Autonomous Agent — Battle City
An intelligent tank agent programmed to navigate a real-time grid environment, dodge enemies, and react to threat states.
* **Algorithms:** BFS, Greedy Best-First Search, A* Search, and Minimax with Alpha-Beta Pruning.
* **Generative Systems:** Integrated with a procedural map generator utilizing Constraint Satisfaction Problems (CSP).
* **Technology Stack:** Python, Pygame.

### 2. Mini_Instagram
A structural console-based replica modeling a core social media backend ledger and feed distribution layer.
* **Implementation:** Designed a custom Hash Map database layer featuring optimized collision resolution to execute rapid registrations, session validations, and timeline insertions.
* **Technology Stack:** C++.

### 3. Soul-Sisters Web App
A full-stack, end-to-end platform constructed to support complete transactional lifecycles and communications.
* **Security:** Configured route-guarding mechanisms utilizing secure stateless JSON Web Tokens (JWT).
* **Technology Stack:** MongoDB, Express, React, Node.js (MERN).

### 4. Connect Four Game
A console-based classic two-player board game written as an early foundation in programming logic.
* **Mechanics:** Engineered horizontal, vertical, and diagonal win-detection array parsing sweeps.
* **Persistence:** Implemented file handling (I/O) to save and reload historical player records.
* **Technology Stack:** C++.

### 5. SkillBridge Interactive Platform
A high-fidelity visual prototype addressing technical skill gaps within modern industry roles.
* **HCI/UX:** Structured scalable UI patterns using dynamic spacing systems, responsive borders, and unified interactive components.
* **Technology Stack:** Figma.
